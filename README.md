

## Getting Started

### To Install and run application:
If you're using docker compose:
```
docker-compose up
```
### OR
if you're using UNIX system without docker compose:
```
npm install
# then run
npm run start-app
```

### OR

```
npm install
# then run
npm run mock-server
# then run
npm run dev
# then
Go to https://localhost:3000
```
